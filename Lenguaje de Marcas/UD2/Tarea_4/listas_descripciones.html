HTML

W3C

World Wide Web Consortium, encargado de desarrollar estándares web.

WHATWG

Web Hypertext Application Technology Working Group, trabaja en el desarrollo de HTML y APIs.

HTML

Lenguaje de Marcado de Hipertexto usado para crear páginas web.

XHTML

Extensible Hypertext Markup Language, versión más estricta de HTML.

DTD

Document Type Definition, define la estructura y los elementos de un documento XML o HTML.

APIs

Interfaces de Programación de Aplicaciones que permiten funcionalidades adicionales en HTML.

Drag and Drop

API que permite arrastrar y soltar elementos en una página web.

Geolocalización

API para obtener la ubicación geográfica del usuario.

Almacenamiento local

Método de almacenamiento de datos en el navegador del usuario.

App Cache

Permite almacenar recursos de una aplicación web para acceso offline.

WebWorkers

API para ejecutar scripts en segundo plano en el navegador.

Editor WYSIWYG

Editor que permite ver el resultado final mientras se edita.

Navegadores

Software usado para navegar en la web.

Mozilla Firefox

Navegador web desarrollado por Mozilla Foundation.

Google Chrome

Navegador desarrollado por Google, conocido por su rapidez y compatibilidad.

Safari

Navegador de Apple para dispositivos iOS y macOS.

Opera

Navegador conocido por varias funcionalidades web.
