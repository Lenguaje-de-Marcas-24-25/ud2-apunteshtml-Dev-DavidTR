¿Qué es el lenguaje HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Define la estructura de una página web mediante etiquetas y elementos.

¿Cuál es la relación de Tim Berners-Lee con HTML?

Tim Berners-Lee es el inventor de HTML. Creó este lenguaje en 1991 como parte de la creación de la World Wide Web, permitiendo la comunicación de información a través de Internet.

¿Cuáles son las distintas versiones del lenguaje HTML?

Las versiones de HTML incluyen HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01 y HTML5, que es la versión más reciente y ampliamente utilizada hoy en día.

¿Qué es la W3C?

El World Wide Web Consortium (W3C) es una organización internacional que desarrolla estándares para asegurar el crecimiento de la web a largo plazo. Ha sido clave en el desarrollo de HTML y otros lenguajes web.

¿Qué es el WHATWG?

El WHATWG (Web Hypertext Application Technology Working Group) es un grupo de trabajo fundado en 2004 por desarrolladores de navegadores para estandarizar el desarrollo de tecnologías web, especialmente HTML.

¿Qué es un Living Standard?

Un Living Standard (Estándar Vivo) es una especificación que se actualiza continuamente, en lugar de lanzar versiones fijas. HTML5 es un ejemplo, ya que se mejora constantemente en lugar de reemplazarse por una nueva versión.

¿Qué es el lenguaje XHTML?

XHTML (Extensible HyperText Markup Language) es una versión más estricta de HTML basada en XML. Fue desarrollado para mejorar la interoperabilidad de los documentos web y hacerlos más consistentes.

¿Cuáles son las distintas versiones del lenguaje XHTML?

Las versiones de XHTML incluyen XHTML 1.0, XHTML 1.1 y XHTML5. XHTML5 es compatible con HTML5, manteniendo las reglas de XML.

¿Cuáles son las diferencias sintácticas y estructurales entre XHTML y HTML?

XHTML es más estricto que HTML en cuanto a sintaxis. En XHTML, todas las etiquetas deben cerrarse, los atributos deben estar entre comillas y es sensible a mayúsculas y minúsculas, mientras que HTML es más flexible en estos aspectos.
