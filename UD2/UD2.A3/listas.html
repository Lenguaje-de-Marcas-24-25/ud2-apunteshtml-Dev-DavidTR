    1. W3C
    World Wide Web Consortium, desarrolla estándares web.
    2. WHATWG
    Web Hypertext Application Technology Working Group.
    3. HTML
    Lenguaje de marcado para estructurar documentos web.
    4. XHTML
    Versión de HTML basada en XML.
    5. DTD
    Define estructura de documentos HTML/XML.
    6. APIs:
    1. Drag and Drop
    Permite arrastrar y soltar elementos.
    2. Geolocalización
    Permite obtener la ubicación geográfica del usuario.
    3. Almacenamiento local
    Guarda datos en el navegador.
    4. App Cache
    Guarda recursos para acceso offline.
    5. Web Workers
    Permite ejecutar scripts en segundo plano.

    Editor de texto visual que permite ver cómo quedará el contenido.
